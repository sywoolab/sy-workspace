## 게이트 우선순위 (R2 지적 반영 — 충돌 시)

**HRV 일일 게이트 > ACWR 주간 게이트 > monotony 월간 게이트.**
단기 신체 신호가 항상 우선한다. HRV 게이트로 강도를 연속 하향해 monotony가 올라가는 경우, 강도가 아니라 **휴식일 삽입**으로 monotony를 해소한다 (저강도 연속으로 풀지 않음).

- HRV 게이트: 가민 야간 HRV 7일 평균이 개인 정상범위 미만 → 당일 질세션을 Easy로 전환. 비정상 상승도 동일 (과훈련 전조).
- ACWR 게이트: 주간(7일)÷만성(28일) > 1.3 → 다음 3일 볼륨 동결. < 0.8 → 점진 증량 허용.
- monotony 게이트: 주간 monotony > 2.0 → 완전 휴식일 1개 삽입.
- 보조 신호: RHR 평소 대비 +10bpm = 당일 Easy/휴식 (기존 마스터 규칙 유지).

## 주기화 캘린더 규칙

- 대회 간격 ≤ 4주 구간("레이스 밀집기"): **유지 주기화** — 볼륨 증량 금지, 테이퍼 반복만.
- 대회 간격 > 4주 구간: **블록 주기화** — 축적(2~4주, 볼륨) → 변환(2~3주, 레이스 강도) → 실현(테이퍼 10일).
- 전환 시점 규칙: 대회 후 회복 1주 종료 시 다음 블록 개시 (예: 6/28 대가야 → 7/6 축적 블록 개시).
- 테이퍼 표준: 기간 10일, 볼륨 -45% (지수적 감소), 강도·빈도 유지, 러닝 볼륨은 -20~40%로 보수적, 브릭 터치 1회 유지.

## 페이싱 규칙 (레이스 실행)

- 수영: CSS 기반 (CSS = 200m ÷ (T400−T200) 초/100m — 400/200 풀 테스트 분기 1회). 레이스는 CSS+5~8초/100m, 첫 200m 의도적 억제 (수영 오버페이스 → 자전거 혈중젖산 +87%, PMC9556684). 드래프팅 적극 (에너지 -18%). 사이팅 6~10스트로크당 1회.
- 자전거: HR ceiling = LTHR의 92~96% (LTHR = 20분 전력 평균 HR). 파워 있으면 NP 85~90% FTP, VI ≤ 1.05. 자전거 오버페이스의 러닝 비용: -5.8% 거리 손실 실측 (Morin 2019, PMC6572577). 마지막 2km 케이던스 90~100rpm.
- 러닝: 첫 1km 목표보다 +15~20초 (젤리렉), 2km부터 목표 진입, 후반 네거티브 시도. 브릭 감속률: 훈련된 선수 1~5%, 미숙련 10~15% (단일 추정값 사용 시 "추정" 라벨 의무 — 7% 등 중간값은 출처 없음, R1 판정).
- 더위: 27°C+ 시 러닝 3~7% 감속 보정 — 근거 위계: 코칭 플랫폼 합의 (peer-review 미확정, 방향성은 Ely 2007 지지). HR/RPE 기준으로 노력 유지, 페이스 목표는 하향.
- T1/T2: 가장 싼 단축 구간. 목표 T1 ≤ 2:00, T2 ≤ 1:30 (현재 대구 실측 T1 4:30 — 연습으로 -2분 가능).

## 데이터 해석 기준선 (2026-06-07 시점, 레드팀 정정 반영)

- 러닝: 명확한 개선 — 브릭 5:40(5/10) → 5:31(6/7). 단독 환산·VDOT 역산은 감속률 가정 의존이므로 "추정" 라벨.
- 수영: 퇴보 아님 — 연속 세션 기준 2:06(4월) → 1:53(5/30) 개선, 6월은 신선 상태 연속 세션 부재로 비교 불가 (R1 판정: "조건 의존적 변동"). OW 경험 5회 (0회 주장은 기각). 질적 이슈는 실재: 훈련 시간 1위(42%) 종목임에도 dead-zone·드릴 비중 커 레이스 조건 자극 부족.
- 자전거: 레이스 실측 33.7km/h(대구)로 목표 충족. 단 40km 레이스 페이스 특화 세션 0회 — 검증 부족.
- 강도 배분 실태: Easy 40.6% / dead-zone(moderate) 45% / 질세션 14.4% — 채택된 피라미드형(Z1 75~80%)과 가장 크게 어긋나는 지점 = **최우선 교정 대상**.
- 부하 관리 실태: 12주 중 5주 ACWR > 1.3 (W13~16, W20), 현재(W23) 1.23 상한 부근 (R1 재계산 — 중복 제거 합산 기준). W16 복합 위기(ACWR 1.91 + monotony 2.16 + HRV 최저 + Readiness 14) 실증 — 게이트 도입의 본인 데이터 근거.
