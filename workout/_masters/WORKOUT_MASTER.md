## 판단 규칙 (코칭 시 반드시 준수)

### 1. 연속 러닝 규칙
- **복귀 2주 이내 (~ 3/30)**: 러닝 연속 2일 금지. 사이에 반드시 수영 또는 휴식
- **복귀 3주차 이후 (3/31 ~)**: 연속 2일까지 허용, 단 둘째 날은 Easy 페이스 (6:00 이상)
- **연속 3일 러닝**: 어떤 시기든 금지
- **브릭 러닝 카운트**: 토요일 브릭 러닝(5km)은 풀 브릭으로 카운트하며, 연속 러닝 규칙에서 독립 러닝 1회로 카운트

### 2. 스케줄 외 운동 시 조정 로직
사용자가 스케줄과 다른 운동을 했을 때 다음 날 판단 기준:

| 오늘 실제 한 운동 | 내일 원래 스케줄 | 판단 |
|------|------|------|
| 러닝 (스케줄 외) | 러닝 | 복귀 2주 내 → 수영으로 대체. 3주 후 → Easy로 강도 낮춰 유지 |
| 러닝 (스케줄 외) | 수영 | 그대로 수영 진행 (회복 효과) |
| 러닝 (스케줄 외) | 자전거 | 그대로 자전거 진행 |
| 수영/자전거 | 러닝 | 그대로 러닝 진행 |

### 3. 컨디션 기반 조정
사용자가 근육통/피로/통증을 보고했을 때:

- **근육통 (DOMS)**: 러닝 → 수영으로 대체, 또는 Easy 페이스로 거리 50% 감소
- **관절/힘줄 통증**: 해당 종목 즉시 중단, 다른 종목으로 대체. 3일 이상 지속 시 병원 권유
- **옆구리 통증 (횡격막 경련)**: 복귀 초기 정상. 운동 중 대처법 안내 (숨 길게 내쉬기 + 아픈 쪽 팔 위로)
- **심박 비정상 (안정시 대비 +20% 이상)**: 오버트레이닝 의심, 해당 일 Easy 또는 휴식

### 4. 우선순위 (충돌 시)
1. **부상 방지** > 스케줄 준수 > 볼륨 달성
2. **러닝 주 3회 확보** > 수영 횟수 > 자전거 횟수
3. 수영 수업 스킵 가능, **러닝은 부상 외 스킵 금지**
4. 주간 러닝 볼륨이 부족하면 수영 1회를 러닝으로 대체

### 5. 응답 시 필수 확인
운동 관련 질문에 답변하기 전 아래를 **모두** 확인:

**Step 1: 기본 확인 (매 질문마다)**
1. **오늘 요일** — `date` 명령어로 확인. 추측 금지
2. **가민 최근 기록** — 최근 3일 운동 이력 확인 (연속 러닝 여부 판단)
3. **복귀 경과일** — 러닝 복귀일(2026-03-16) 기준 몇 주차인지 계산
4. **현재 Phase** — Phase 1/2/3 중 어디인지

**Step 2: 3-에이전트 + 레드팀 검증 (판단이 포함된 답변 시)**
운동 스케줄 변경, 대회 참가 판단, 훈련 강도 조정 등 **판단이 수반되는 질문**에는 반드시 적용:
- Agent A: 마스터 파일 규칙 기반 판단
- Agent B: 독립적으로 동일 질문에 대한 판단
- Agent C: 외부 자료(웹 검색 등) 기반 교차 검증
- Red Team: A/B/C 결과의 모순·누락·리스크 점검
- 3+1 결과 일치 시 답변. 불일치 시 차이점 보고 후 사용자 판단 요청

### 6. 피로 관리
- **일요일 완전 휴식**: 복귀 초기 근골격계 부담 경감 및 부상 방지를 위해 매주 일요일 완전 휴식
- 수영(월/수/금)이 **능동적 휴식** 역할 (심폐 유지 + 러닝 근육 회복)
- 피로 누적 느껴지면 **월요일 수영을 추가 완전 휴식으로 대체** (가장 적합한 날)
- 주 1회까지 수영 → 휴식 대체 허용. 주 2회 이상 대체 시 수영 볼륨 부족

### 7. 목요일 회식 (고객 저녁) 대응
- 매주 목요일 고객 저녁 → 음주 + 늦은 귀가 가능성
- **금요일 수영 수업: 선택** (컨디션 따라 스킵 가능)
- **토요일 브릭: 절대 스킵 금지** — 금요일 쉬더라도 토요일은 반드시 실행
- 목요일 러닝은 회식 전(아침/점심)에 미리 소화하는 것이 이상적

### 8. 일반 원칙
- 자전거 → 수영 순서 (회복 효과). 역순 비권장
- 대회 전 오픈워터 경험 확보: 5/9(토) 수성못 사전 입수 (대회 전날, 200~300m)
- Phase 1에서 페이스는 신경 쓰지 않는다. 빈도와 거리만 확보
- Phase 2부터 화요일 템포런에서만 페이스 관리

### 9. 수동 운동 업데이트 프로토콜

사용자가 운동을 보고하면 아래 순서로 즉시 처리한다. **기존 가민 자동싱크(12:17/21:17)와 독립적으로 실행**하며, 자동싱크는 그대로 유지한다.

1. **가민 데이터 수집**: 가민 커넥트 API에서 오늘 활동 데이터를 직접 조회 (토큰 기반 로그인, `workout/data/garmin_tokens/`)
2. **가민 미동기화 시**: 사용자에게 워치→앱 동기화 요청 후 재시도
3. **workout_log.json 업데이트**: 가민 메트릭(거리, 페이스, 심박, TE, 부하 등)을 포함한 완전한 엔트리 작성
4. **git push**: push하여 `workout-update.yml` → `workout_analysis.py` → 텔레그램 분석 자동 발송
5. **빈 메트릭 push 금지**: 가민 데이터 없이 기본 정보만으로 push하지 않는다 (0짜리 분석 방지)

### 10. 체감 vs 데이터 동시 검증 (필수)

사용자가 운동 후 주관적 평가("힘들었다", "여유있었다", "심박 안 떨어졌다")를 보고하면 **반드시 가민 객관 지표와 교차 비교**하여 둘 다 보고한다.

- **체감만 보고 판단 금지**: 4/25 사례 — 사용자 "심박 안 떨어져 힘들었다" → 실제 평균 HR 161로 3/1 모의(169) 대비 -8bpm. 체감과 데이터가 괴리될 때 데이터가 더 정확한 경우 많음
- **체감/데이터 괴리 시**: 둘 다 명시적으로 보고하고 가능한 원인(숙취·수면·기온·심리 등) 함께 제시
- **숙취·음주 다음날 HR**: 평소 페이스 대비 평균 HR +5~15bpm 인플레이션. 동일 페이스 환산 시 효율은 정상에 가까울 수 있음

### 11. 자전거 외부 코스 보정 방법론

실외 자전거 데이터를 본대회 환산할 때 다음 절차로 보정:

1. **신호 대기 영향**: 첫 1~2 lap 속도가 다른 lap 대비 5km/h 이상 낮으면 신호/스타트 영향으로 분류, 환산에서 제외
2. **업힐/다운힐**: 반환점 또는 코스 중간 lap 속도가 평균 ±3km/h 이상 벗어나면 지형 영향으로 표기, 양쪽을 평균하여 평지 추정
3. **바람 양방향 평균법**: 갈 때(전반) lap 평균 vs 올 때(후반) lap 평균을 산출. 두 평균의 산술 평균이 무풍 환산 속도 (단순 1/2 + 1/2 평균이 아닌 시간 가중 평균 권장)
4. **사용자 자기 추정 검증**: 사용자가 "무풍이면 X km/h 가능"이라고 주장할 때 위 데이터 기반 보정값과 비교. 괴리 시 데이터 기반 보수적 추정 채택

**4/25 적용 예**: 실측 29.2km/h → 신호+업힐 보정 후 양방향 평균 ≈ 31.3km/h → HR 여유분 변환 시 31.5~32km/h가 본대회 상한 추정. 사용자 주장 34km/h는 데이터 미지지.

### 12. 회복식 가이드라인

운동 직후~수 시간 윈도우 식사 추천 시:

- **글리코겐 회복 우선**: 고부하 운동(TL 200+) 후엔 고GI 탄수도 권장 (떡, 흰밥, 바나나 등). 평소 기준의 "정제탄수 회피"는 적용하지 않음
- **단백질 동반**: 탄수만으론 근손실 방지 부족. 계란/닭/두부/요거트 등 20~40g 단백질 보충
- **나트륨 누적 관리**: 외식 메뉴 추천 시 일일 누적량 추정. 피자(2,000~3,000mg) + 김치찌개 국물(2,000~2,500mg) = 위험. 한 가지가 이미 고나트륨이면 다른 한 가지는 저나트륨으로 균형
- **늦은 시간(21시 이후)**: 캡사이신·국물·고지방 회피. 위 운동성 저하 + 수면 단절 + 체수분 저류
- **식사 마감 시간**: 취침 2시간 전. 그 이후엔 가벼운 단백질만 (우유/요거트)

- **공통 원칙**: 페이스가 아닌 심박·체감으로 조절. 통증/어지럼증 시 즉시 중단
- **수분·전해질 보충**: 운동 1시간 전부터 500ml 이상

### 14. 수영 풀 선택 기준 (50m vs 25m)

| 상황 | 추천 |
| ------ | ------ |
| 전일 고부하(TL 300+), 회복 세션 | 25m (이동 비용 최소화) |
| 본대회 페이스 시뮬, 컨디션 정상 | 50m (오픈워터 시뮬 적합) |
| 1km 이하 단축 세션 | 25m |
| 1.5km+ 연속 페이싱 시뮬 | 50m (가능 시) |
| 이동 1시간 이상 + 컨디션 회복 필요 | 25m |

- **25m에서 오픈워터 시뮬 보강**: 벽 푸시오프 짧게, 양쪽 호흡, 분할 금지(연속)
- **50m 가는 경우**: 본대회 2주 전 1회로 충분. 매주 갈 필요 없음

### 15. 오픈워터 첫 적응 가이드 (5/9 테스트 대비)

오픈워터 첫 경험 시 반드시 안내할 5가지:

1. **사이팅 리듬**: 6~10스트로크마다 1번 정면 부표 확인
2. **워밍업 필수**: 입수 후 얼굴 담그고 호흡 30초~1분 (저온 패닉 방지)
3. **수트 입수 직전 물 흘려넣기**: 수트-몸 단열층 형성
4. **출발 인파 회피**: 라인업 끝 또는 5~10초 늦게 출발
5. **호흡 양쪽**: 풀에서도 양쪽 호흡 연습 (파도/햇빛 방향 대응)

**페이스 환산**: 풀 페이스 + 10~30초/100m가 오픈워터 페이스 (사이팅·수트·인파·파도 영향). 첫 OW에서 풀 페이스 그대로 안 나오는 게 정상.
